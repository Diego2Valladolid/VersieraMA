Versiera Meets Arcs - Version 98 
A geometrical meditation on the shape of breasts after Maria Gaetana Agnesis Versiera curve aka "The Witch".
Runs as a plain html page 
Invokes SageMathCell 
Tested with Chrome 

User manual
Install and launch the html page as "index.html"
Click on the button "Let there be breasts"
Move the slider at the top of the page to change nipple coordinates.

Known issues 
- the slider may not appear in Internet Explorer
